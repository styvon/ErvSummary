# Summarising ERV

## Usage
`main.py -e <ervdir> -r <refdir> -o <outputdir> -n <nmer> -c <center>`  

Example: `python code/main.py -e data/files_erv -r data/files_ref -o data -n 5 -c 4`  
counts 5-mer patterns from 9mer files
